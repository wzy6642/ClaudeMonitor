# Claude Monitor

一个轻量级的 Windows 桌面工具，用于监控 Claude Code 实例的运行状态。通过类似苹果灵动岛的 UI 设计，让用户无需切换窗口即可掌握所有实例状态。

## 功能特性

- **实时状态监控** - 检测 Claude Code 的运行状态
  - `RUN` - 模型推理中（深天蓝色）
  - `WAIT!` - 需要用户确认（橘黄色 + 声音提醒）
  - `DONE` - 任务完成（绿色）
  - `ERR` - 错误状态（红色）

- **灵动岛 UI** - 顶部居中显示，圆角矩形设计
- **系统托盘** - 右键最小化到托盘，后台运行
- **不透明度调节** - 25% / 50% / 75% / 90% / 100%
- **配置持久化** - 自动保存设置，下次启动自动加载

## 系统要求

- Windows 10 或更高版本
- Windows 11 可享受圆角窗口效果

## 使用方法

### 基本操作

| 操作 | 功能 |
|------|------|
| 左键点击灵动岛 | 跳转到对应的 Claude Code 窗口 |
| 右键点击灵动岛 | 最小化到系统托盘 |
| 双击托盘图标 | 显示窗口 |
| 右键托盘图标 | 打开菜单 |

### 托盘菜单

- **显示窗口** - 恢复显示灵动岛
- **不透明度** - 调整窗口透明度
- **退出** - 关闭程序

## 安装与运行

1. 双击 `ClaudeMonitor.exe` 即可运行
2. 首次运行会在同目录下生成 `config.json` 保存设置

## 常见问题

**Q: 为什么检测不到 Claude Code 窗口？**

A: 确保 Claude Code 正在运行，且窗口标题包含 "Claude Code" 或以 spinner 字符开头。

**Q: 如何完全退出程序？**

A: 右键点击托盘图标，选择"退出"。

**Q: 为什么没有圆角效果？**

A: 圆角效果仅支持 Windows 11，Windows 10 会显示普通矩形窗口。
